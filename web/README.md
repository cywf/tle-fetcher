# TLE Pass Dashboard

A static Next.js web application for computing satellite pass predictions (line-of-sight windows) using SGP4 propagation.

## Features

- **Retro Synthwave UI**: Neon-themed control panel with analog gauges, LED indicators, and glowing effects
- **Real-time Pass Computation**: Client-side SGP4 propagation using satellite.js
- **TLE Data Sources**: Local assets with CelesTrak API fallback
- **Satellite Metadata**: Fetches from SatNOGS DB and CelesTrak SATCAT
- **Fully Static**: Runs on GitHub Pages without server dependencies
- **Privacy-Friendly**: No API keys required for basic functionality

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Lint code
npm run lint
```

The development server runs at: http://localhost:3000/tle-fetcher/

## Architecture

### Pages
- `pages/_app.tsx` - Application wrapper
- `pages/index.tsx` - Main dashboard page

### Components
- `components/ControlPanel.tsx` - Input form for location, time, and parameters
- `components/ResultsTable.tsx` - Display of computed passes with sorting
- `components/SatDetailsModal.tsx` - Metadata popup for satellites
- `components/Gauges.tsx` - Analog status indicators

### Libraries
- `lib/passes.ts` - SGP4 propagation and LOS computation
- `lib/tle.ts` - TLE loading (local + remote fallback)
- `lib/metadata.ts` - Satellite metadata fetching
- `lib/time.ts` - Date/time parsing and UTC conversion

## How It Works

1. **User Input**: Observer location (lat/lon), date/time, and time-on-target (ToT)
2. **TLE Loading**: Fetches Two-Line Element sets from local assets or CelesTrak
3. **SGP4 Propagation**: Computes satellite positions at 10-second intervals
4. **Visibility Check**: Determines when elevation exceeds minimum threshold (default 5Â°)
5. **Results**: Displays satellites visible during the window with pass details

## Configuration

The site is configured for GitHub Pages deployment:

- **basePath**: `/tle-fetcher` (adjust in `next.config.js`)
- **output**: `export` (static HTML generation)
- **images**: `unoptimized: true` (for static hosting)

## Data Sources

- **TLE Data**: 
  - Primary: Local assets (`public/tle/*.tle`) generated by CI
  - Fallback: CelesTrak GP/TLE API
- **Metadata**:
  - SatNOGS DB API: https://db.satnogs.org/api/satellites/
  - CelesTrak SATCAT: https://celestrak.org/satcat/
